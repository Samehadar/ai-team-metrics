# Example Data

Real Cursor IDE usage data from a 14-person engineering team, anonymized.

## How to use

1. Start the dashboard: `npm run dev`
2. Open http://localhost:5173
3. Drag & drop all CSV and JSON files into the upload area

## Files

14 developers × 2 formats (CSV + JSON) = 28 data files.

Upload CSV files to see Overview, Adoption, Timeline, and Models tabs.
Upload JSON files to unlock the Code Impact tab (lines of code, accept rates, languages).
